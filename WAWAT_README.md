# Wawat Flutter Application

## Описание

Wawat - это приложение для поиска курьеров для доставки посылок. Приложение позволяет пользователям искать курьеров, которые летят между городами, и передавать с ними посылки.

## Структура проекта

```
lib/
├── wawat/
│   ├── modals/                   # Модальные окна
│   │   ├── auth_required_modal.dart    # Модал "Требуется регистрация"
│   │   ├── login_modal.dart            # Модал входа
│   │   └── registration_modal.dart     # Модал регистрации
│   ├── screens/                  # Экраны приложения
│   │   ├── home_screen.dart            # Главный экран с поиском
│   │   └── search_results_screen.dart  # Экран результатов поиска
│   ├── widgets/                  # Общие виджеты
│   │   ├── wawat_button.dart           # Кнопки
│   │   ├── wawat_chip.dart             # Чипы/теги
│   │   ├── wawat_courier_card.dart     # Карточка курьера
│   │   └── wawat_input_field.dart      # Поля ввода
│   └── wawat_app.dart            # Главный файл приложения
└── presentation/
    └── resourses/
        ├── wawat_colors.dart     # Цветовая схема
        ├── wawat_text_styles.dart # Текстовые стили
        └── wawat_dimensions.dart  # Размеры и отступы
```

## Реализованные функции

### 1. Главный экран (Home Screen)
- ✅ Header с логотипом и иконкой уведомлений
- ✅ Hero секция с градиентом и иконкой поиска
- ✅ Форма поиска с 5 полями:
  - Откуда (город отправления)
  - Куда (город назначения)
  - Дата с
  - Дата по
  - Категория (Все, Курьеры, Отправители, Покупатели)
- ✅ Кнопка поиска с градиентом
- ✅ Секция "Популярные предложения" с карточками курьеров
- ✅ Bottom Navigation с 5 табами

### 2. Модальные окна
- ✅ **Требуется регистрация** - показывается при попытке доступа к защищенным функциям
- ✅ **Регистрация** - форма с полями:
  - Полное имя
  - Email
  - Телефон
  - Пароль
  - Подтверждение пароля
  - Языки общения
  - Checkbox согласия с условиями
- ✅ **Вход** - форма входа с email и паролем

### 3. Экран результатов поиска
- ✅ Header с кнопками "Фильтры" и "На карте"
- ✅ Информация о поиске с активными фильтрами
- ✅ Сортировка результатов (по рейтингу, цене, дате)
- ✅ Список найденных курьеров

### 4. UI компоненты
- ✅ **WawatButton** - кнопки с градиентом и outline вариантом
- ✅ **WawatInputField** - поля ввода с иконками и валидацией
- ✅ **WawatDropdownField** - выпадающие списки
- ✅ **WawatChip** - чипы/теги разных типов
- ✅ **WawatCourierCard** - карточка курьера с полной информацией

Приложение использует именованные маршруты:
- `/` - Главный экран (WawatHomeScreen)
- `/search-results` - Экран результатов поиска (SearchResultsScreen)

## Интерактивность

### Home Screen:
1. Клик на поля формы → открывает соответствующие picker'ы (TODO)
2. Клик "ПОИСК" → переходит на страницу результатов
3. Клик на "Написать" в карточке курьера → показывает модал требования входа
4. Клик на иконку "Аккаунт" в bottom nav → показывает модал требования входа

### Модалы:
1. Клик "Зарегистрироваться" → открывает модал регистрации
2. Клик "Войти" → открывает модал входа
3. Переключение между модалами через ссылки внизу
4. Валидация полей формы
5. Показ/скрытие пароля

### Результаты поиска:
1. Кнопка назад → возврат на главный экран
2. Клик на фильтр chip → удаление фильтра (TODO)
3. Сортировка через dropdown

## Разработка

При разработке новых функций следуйте структуре:

1. **Создание нового экрана**:
   - Создайте файл в `lib/wawat/screens/`
   - Добавьте маршрут в `wawat_app.dart`

2. **Создание нового компонента**:
   - Создайте файл в `lib/wawat/widgets/`
   - Используйте существующие цвета и стили из `presentation/resourses/`

3. **Добавление модала**:
   - Создайте файл в `lib/wawat/modals/`
   - Реализуйте статический метод `show()` для удобного вызова
